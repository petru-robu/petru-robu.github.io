Acorduri

Acordul este o combinație de cel puțin trei note muzicale interpretate simultan.

  • in game majore: I, ii, iii, IV, V, vi, vii°
  • in game majore: I(M7), ii(7), iii(7), IV(M7), V(dom7), vi(7), vii°(min7b5)

  • in game minore: i, ii°, III, iv, v, VI, VII
  • in game minore: i(7), ii°(min7b5), III(M7), iv(7), v(7), VI(M7), VII(dom7)

    • Ordinea diezilor este: Fa, Do, Sol, Re, La, Mi, Si
    Ordinea bemolilor este: Si, Mi, La, Re, Sol, Do, Fa
    Relativa minora: o terta mai jos (ex: Gmaj~Emin, Emaj~C#min, etc.)
